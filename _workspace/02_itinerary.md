## 3. 동선 최적화 (route-optimizer 적용)

### 적용한 5대 원칙

```
1. 클러스터링: 정문·글로벌페어·포시즌스·차이나문·포시즌스베이비서비스를
   "포시즌스 클러스터"로 묶어 오전 80% 활동 집중 (도보 5분 이내)
2. 순환 경로: 정문 → 포시즌스 → 차이나문 → 베이비서비스 → 사파리 →
   로스트밸리 → 매직랜드 → 정문 — 완전 단방향 흐름 (지그재그 0)
3. 시간 정합: 사파리월드 10:30 개장·로스트밸리 17:00 마감 모두 반영,
   페스티벌 트레인 17:00 마감 전 11:00 탑승
4. 피크 회피:
   - 포시즌스 튤립 단체 진입(11:30) 전 10:00~11:30 사진·트레인 완료
   - 점심 피크(12:00~13:30) 회피 11:30 얼리 런치
   - 사파리 단체 빠진 후 14:30 진입
5. 여유 확보: 총 이동시간 도보 약 50분 (체류 7시간의 12%) — 권장 25% 이내 충분
```

### 클러스터별 시간 배분

| 클러스터 | 위치 | 체류시간 | 활동 | 도보 거리 |
|---------|------|---------|------|----------|

## 10. 핵심 원칙 재확인 (현장에서 지킬 것)

> 🎯 **"무리 없는 일정 > 영유아 안전·편의 > 어트랙션"**

1. **시간에 쫓기지 않기** — 30개월 낮잠 시간이 길어지면 사파리월드 14:30 → 15:00 늦춰도 OK. 사파리는 18:00까지 운영.
2. **포시즌스 베이비서비스를 베이스캠프로** — 짐 일부 보관, 식수, 낮잠, 기저귀 모두 한 곳에서 처리.
3. **14개월 컨디션 1순위** — 어떤 어트랙션도 14개월 안전·수면보다 우선하지 않음. 16:00 조기 출차 가능성 항상 열어두기.
4. **30개월 1순위 라이드 = 페스티벌 트레인 + 사파리월드 + 로얄쥬빌리캐로셀 + 로스트밸리** — 이 4개만 타도 충분히 만족.
5. **부부 교대 휴식** — 12:30~14:30 낮잠 블록은 부모도 진짜 휴식 시간. 한 명은 침대 옆 대기, 한 명은 카페 30분.
6. **저녁 러시아워 회피** — 17:30 출차로 19:00 도착 → 영유아 취침 루틴 안정.
7. **야간 운영 비참여** — 일루미네이션·불꽃놀이·퍼레이드 모두 패스. 영유아 동반에 야간은 부적합.
8. **돌발 시 즉시 자차 복귀 옵션** — 정문 도보 5분 + 카시트 익숙 환경 = 영유아의 안전망.
