# 00. 사용자 입력 정리

## 기본 정보

| 항목 | 내용 |
|------|------|
| 출발지 | 서울 여의도 |
| 목적지 | **에버랜드 (경기 용인) 확정** |
| 여행 형태 | 당일치기 (1day) |
| 여행 일자 | 2026-04-28 (화) 평일 기준 — 사용자 별도 지정 시 변경 |
| 동행 | 부부 + 30개월 아기 + 14개월 아기 (총 4인, 영유아 2명) |
| 이동수단 | 자차 (카시트 2개 구비) |
| 출발 시간 | 이른 아침 (6~7시) |
| 예산 | 고려하지 않음 — 무리 없는 일정이 최우선 |
| 테마 | 아이 체험 (동물원·목장·아쿠아리움) + 자연 산책 (수목원·공원) + 에버랜드 |

## 핵심 제약 / 설계 원칙

1. **무리 없는 일정** — 관광지 1~2곳 집중, 휴식·낮잠 시간 충분히 확보
2. **영유아 2명 동반** —
    - 30개월: 보행 가능, 호기심 왕성, 점심 후 1회 낮잠
    - 14개월: 걸음마 단계, 오전·오후 2회 낮잠 필수, 분유·수유 필요
3. **자차 활용** — 짐 적재 자유, 카시트 낮잠·기저귀 갈이·짐 보관 가능
4. **수유실·기저귀 교환대 접근성**이 동선의 축
5. **귀가 시 영유아 취침 루틴 보장** — 19시 이전 도착 권장

## 우선순위

1. 안전·편의 (수유실, 기저귀, 응급 대응)
2. 영유아가 즐길 수 있는 코너 큐레이션
3. 부모의 휴식 시간 확보 (커피·간식·잠시 앉기)
4. 비용·럭셔리 요소는 후순위
